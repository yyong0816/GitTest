브랜치(branch)
- 새로운 작업을 수행하거나
- 분리된 작업 환경을 제공

main(or master) branch
- 저장소 생성시 가장 기본이 되는 브랜치

1. 새로운 브랜치를 생성해서 작업환경 분리
2. 각 브랜치 별로 버전을 별도로 관리
3. 작업이 끝난 브랜치는 main에 합쳐서 적용 가능


명령어)
git branch
- 현재 저장소에 만들어진 브랜치 목록을 출력

git branch [이름]
- 지정 이름으로 현재 HEAD 부분에서 새 브랜치 생성

git branch -d [이름]
- 지정 브랜치를 제거

git checkout [이름]
- 지정 브랜치로 이동

git branch [이름] [커밋해시]
- 지정 버전 위치에 브랜치를 생성
