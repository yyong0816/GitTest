버전 되돌리기(reset)
1. git reset --soft [커밋해시]
- 지정 버전으로 HEAD를 이동
- 단, 파일은 되돌아가지 않고 스테이징된 상태로 남는다

2. git reset --mixed [커밋해시]
- 지정 버전으로 HEAD를 이동
- 단, 파일은 되돌아가지 않고 스테이징도 해제된다

3. git reset --hard [커밋해시]
- 지정 버전으로 HEAD를 이동
- 앞의 버전은 모두 제거되며 변경된 파일도 모두 되돌아간다

※ hard는 실제 파일을 복구 용도로 사용하고
※ soft/mixed는 파일은 남기고 버전(commit)만 제거하는 용도

즉, soft/mixed는 너무 많아진 버전 정보를 줄이기 위해서 사용
